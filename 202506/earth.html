地学 定期テスト対策問題 (高校2年)

出題範囲: 地球の概観 (地球の形・大きさ・内部構造、重力、地震波など)

試験時間: 60分

満点: 100点

注意事項

  • 解答は、指定された箇所に明確に記入すること。
  • 計算問題では、計算過程も示すこと。特に指示がない限り、有効数字は考慮しなくて良いが、常識的な範囲で答えること。
  • 図やグラフを用いる問題では、それらを正確に読み取ること。

第1問 次の各問いに答えなさい。(知識・技能、計20点)

以下の文中の ( ア ) 〜 ( オ ) に最も適する語句を答えなさい。(各2点×5 = 10点)

平均海水面を陸地にまで延長したと仮定した、地球の理論的な形を ( ア ) といい、これは重力の ( イ ) 面と一致する。

地震波のうち、固体・液体・気体のいずれも伝わる縦波を ( ウ ) 波といい、固体のみを伝わる横波を ( エ ) 波という。

地球の外核が液体であるために、( エ ) 波が伝わらず、震源の反対側に大きく広がる領域を ( オ ) という。

ヒントを見る

(ア) 海水面の形が基準です。

(イ) 重力が等しい面のことです。

(ウ)(エ) 地震波の基本的な種類です。縦波か横波か、伝わるものの状態を思い出しましょう。

(オ) S波が伝わらないことでできる「影」の部分です。

解答を見る

(ア) ジオイド

(イ) 等ポテンシャル

(ウ) P

(エ) S

(オ) シャドーゾーン (またはS波シャドーゾーン)

(1) 地球楕円体

(2) アイソスタシー

ヒントを見る

(1) 地球の形を近似する回転楕円体のことです。

(2) 地殻とマントルの間の力のつり合いです。

解答を見る

(1) 地球の形に最も近い回転楕円体で、測地計算の基準面として用いられる。

(2) 地殻とマントルの間で、地殻の高さや密度に応じて浮力が働き、重力的な均衡が保たれている状態または考え方。

以下の図は、地球内部の地震波速度の分布を示したものである。A、B、Cで示される不連続面の名称をそれぞれ答えなさい。(各2点×3 = 6点)

第2問 地球の形と大きさと重力に関する次の問いに答えなさい。(思考・判断・表現、計30点)

古代ギリシャのエラトステネスは、地球の大きさを以下のような方法で測定したと言われている。
夏至の日の正午、シエネの町 (現在のエジプト・アスワン) では太陽は天頂にあり、深い井戸の底まで太陽光が届いた。一方、シエネのほぼ真北にあるアレクサンドリアでは、同じ日時の太陽の南中高度は天頂から 7.2° であった。2都市間の距離を 5000 スタジア (約925 km) と仮定する。

(1) 地球が球形であると仮定した場合、この情報から地球の全周長および半径はそれぞれ何kmと計算されるか。円周率は3.14として計算しなさい。(計算過程も示すこと) (8点)

(2) この測定方法にはいくつかの誤差要因が考えられる。どのような要因が誤差を生む可能性があるか、2つ挙げなさい。(各3点×2 = 6点)

ヒントを見る

(1) 地球の全周に対する2都市間の弧の角度 (7.2°) と、その弧の実際の長さ (925 km) の比を使います。360° : (地球の全周長) = 7.2° : 925 km。半径は「全周長 = 2 × 円周率 × 半径」から求めます。

(2) シエネとアレクサンドリアが正確に南北にあるか、距離の測定は正確か、太陽光が完全に平行とみなせるか、など。

解答を見る

(1) 計算過程:

地球の全周長を L [km]、半径を R [km] とする。

中心角の比と弧の長さの比は等しいので、

7.2° : 360° = 925 km : L km

L = (360° / 7.2°) × 925 km = 50 × 925 km = 46250 km

全周長 L = 2πR より、

R = L / (2π) = 46250 km / (2 × 3.14) = 46250 km / 6.28 ≒ 7364.6... km

解答: 全周長: 46250 km、半径: 約 7365 km (または問題の指示に応じて丸める)

(2) 誤差要因の例:

  • アレクサンドリアとシエネが正確に同一経線上にない可能性。
  • 2都市間の距離測定の不正確さ。
  • 太陽光線が完全に平行であるという仮定のズレ (地球と太陽の距離は有限であるため)。
  • 角度測定の不正確さ。
  • 地球が完全な球形ではないこと。

重力異常について、次の問いに答えなさい。

(1) フリーエア異常とブーゲー異常は、それぞれどのような補正を行った重力異常か。また、それぞれ主に何を示唆しているか簡潔に説明しなさい。(各4点×2 = 8点)

(2) ある山脈地域でブーゲー異常を測定したところ、顕著な負の値を示した。このことから、この山脈の地下構造についてどのようなことが推測されるか。「アイソスタシー」という語句を用いて説明しなさい。(8点)

ヒントを見る

(1) 「フリーエア補正」は高さの影響のみ、「ブーゲー補正」は高さと観測点と基準面の間の物質の引力の影響を補正します。フリーエア異常は広域的な地下構造の不均一、ブーゲー異常はより局所的な密度の不均一を示唆します。

(2) 山脈のような地形的な高まりの下には、アイソスタシーを保つために、マントルよりも密度の小さい地殻の「根」が存在すると考えられます。密度の小さい物質が多ければ、その場所の物質による引力は小さくなり、標準重力よりも測定値が小さくなるためブーゲー異常は負になります。

解答を見る

(1)

フリーエア異常:
補正: 観測点の標高に対する補正 (フリーエア補正) のみを行った重力異常。
示唆: 広範囲の地下の質量の過不足やマントルの不均質構造など、大規模な地下構造を反映する。

ブーゲー異常:
補正: フリーエア補正に加え、観測点とジオイド面の間の物質の引力 (ブーゲー補正) と地形補正を行った重力異常。
示唆: 地殻浅部の密度構造の不均一性や、アイソスタシーの状態を反映する。

(2) ブーゲー異常が顕著な負の値を示したことは、その山脈の地下には周囲よりも密度の小さい物質が厚く存在することを示唆する。アイソスタシーの考え方に基づくと、高い山脈を支えるために、その下部にはマントル中に地殻物質が根のように深く入り込んでいる (山の根) と推測される。この密度の小さい地殻の根が存在することにより、その場所の重力が小さくなり、ブーゲー異常が負となる。

第3問 地震波と地球内部構造に関する次の問いに答えなさい。(思考・判断・表現、計30点)

右の図は、ある地震のP波とS波の走時曲線である。以下の問いに答えなさい。

ここにP波とS波の走時曲線が表示されます。

(横軸: 震央距離、縦軸: 地震波の到達時間(走時))

(1) P波とS波を区別する特徴を、伝わる速さと振動方向の観点からそれぞれ述べなさい。(4点)

(2) この図から、P波の速度とS波の速度をそれぞれ求めなさい。(グラフから適切な値を読み取り、計算過程も示すこと) (6点)

(3) 震央距離が200kmの地点で、初期微動継続時間 (P波が到着してからS波が到着するまでの時間) は何秒か。(4点)

(4) もし、地球内部の物質が深さによらず均質であった場合、走時曲線はどのような形になると考えられるか。その理由とともに説明しなさい。(6点)

ヒントを見る

(1) P波は速く、S波は遅い。P波は進行方向に平行に振動する疎密波、S波は進行方向に垂直に振動するねじれ波。

(2) 走時曲線の傾きが「時間/距離」なので、その逆数が速度「距離/時間」になります。グラフから読み取りやすい2点を選んで計算しましょう。

(3) P波の到着時間とS波の到着時間をグラフから読み取り、その差を求めます。

(4) 均質ならば地震波の速度は一定です。速度が一定の運動の「距離と時間の関係グラフ」はどのような形になるか思い出しましょう。

解答を見る

(1)
伝わる速さ: P波はS波よりも速く伝わる。
振動方向: P波は波の進行方向に平行に振動する縦波(疎密波)であり、S波は波の進行方向に垂直に振動する横波(ねじれ波)である。

(2) (グラフがないため、具体的な数値は仮定して計算方法を示します)
例: P波が震央距離 400km を 50秒で到達する場合、P波の速度 Vp = 400km / 50s = 8.0 km/s。
例: S波が震央距離 400km を 100秒で到達する場合、S波の速度 Vs = 400km / 100s = 4.0 km/s。

(3) (グラフがないため、(2)の仮定値を用いて計算方法を示します)
震央距離 200km の地点でのP波の到達時間 Tp = 200km / 8.0km/s = 25秒。
震央距離 200km の地点でのS波の到達時間 Ts = 200km / 4.0km/s = 50秒。
初期微動継続時間 = Ts - Tp = 50秒 - 25秒 = 25秒。

(4) 地球内部の物質が均質であれば、地震波の速度は深さによらず一定となる。そのため、走時曲線は原点を通る直線となる。理由は、速度が一定の場合、距離と時間は比例関係になるからである。

P波のシャドーゾーン (103°～143°) とS波のシャドーゾーン (103°～103°) が存在することから、地球内部の構造について何がわかるか。特に外核の状態に着目して説明しなさい。(10点)

ヒントを見る

S波が外核を伝わらないこと、P波が外核とマントルの境界で屈折・減速することから、外核が液体であると推定できます。S波のシャドーゾーンがなぜ103°以降全てなのか、P波のシャドーゾーンがなぜリング状にできるのかを説明しましょう。

解答を見る

S波は横波であり液体中を伝わることができない。S波のシャドーゾーンが角距離103°以降の広範囲に存在することから、地球内部の深さ約2900kmより深い部分 (外核) が液体状態であることがわかる。P波は縦波なので液体中も伝わるが、マントルから液体の外核に入射する際に速度が急激に低下し、大きく屈折するため、角距離103°から143°の範囲には直接到達できずP波のシャドーゾーンが生じる。しかし、外核を通過し、さらに内核との境界で屈折・反射したP波が143°以降に再び観測されることから、外核の存在と液体であることが強く示唆される。また、内核は固体であることも推定される。

第4問 地球の概観に関する総合的な考察問題に答えなさい。(思考・判断・表現、計20点)

地球内部の構造は、地震波の観測や重力測定など、直接見ることのできない情報から推定されている。もし、地球とは異なる、ある未知の惑星Xが存在すると仮定する。この惑星Xについて以下の2つの観測結果が得られた。

観測結果1：惑星Xの表面で発生した地震を観測したところ、P波は惑星全体に伝わるが、S波は震源の反対側約90°以降の領域には伝わらないことがわかった。

観測結果2：惑星Xの平均密度は地球の平均密度よりも小さいが、地殻を構成する岩石の密度は地球の地殻岩石とほぼ同じであった。また、惑星Xの赤道付近では、遠心力の影響を除いても、極付近に比べて重力がやや小さい傾向が見られた。

これらの観測結果から、惑星Xの内部構造 (特に核の状態やマントルの特徴) についてどのようなことが推測できるか。地球と比較しながら、あなたの考えを具体的に述べなさい。根拠となる観測結果のどの部分からそう判断したのかも明確にすること。(20点)

ヒントを見る

観測結果1から、惑星Xの核の一部がS波を通さない状態、つまり液体である可能性が推測できます。S波のシャドーゾーンの広がり方から、その液体核の相対的な大きさを地球と比較して考察してみましょう。

観測結果2の前半（平均密度と地殻密度）から、マントルや核の密度が地球と比べてどうであるか推測できます。後半（赤道付近の重力）は、地球と同様に自転による遠心力だけでなく、赤道方向にマントル物質の何らかの偏りがある可能性を示唆しているかもしれません（例：赤道バルジが地球より顕著である、あるいはマントル対流のパターンなど）。

これらの情報を組み合わせて、惑星Xの内部が地球とどう異なり、どう似ているかを論理的に説明しましょう。

解答を見る

惑星Xの内部構造についての推測：

1. 核の状態と大きさ (観測結果1より): S波が震源の反対側約90°以降に伝わらないということは、惑星Xにも液体状態の核 (外核に相当する部分) が存在すると考えられる。地球のS波シャドーゾーンは103°以降であるのに対し、惑星Xでは90°以降とより広範囲である。これは、惑星Xの液体核が惑星全体の半径に対して地球の液体核よりも相対的に大きいことを示唆している。P波は全体に伝わることから、地球と同様に固体内核が存在する可能性もあるが、この情報だけでは断定できない。

2. マントルと核の密度 (観測結果2前半より): 惑星Xの平均密度は地球より小さいが、地殻の密度は地球とほぼ同じである。これは、惑星Xのマントルおよび核の平均密度が、地球のそれらよりも小さいことを意味する。特に、相対的に大きな液体核を持つにもかかわらず平均密度が小さいことから、核を構成する物質が地球の鉄・ニッケル主体の核よりも軽い元素を多く含んでいるか、あるいはマントルの密度が地球のマントルよりも顕著に小さい可能性が考えられる。

3. マントルの不均質性または形状 (観測結果2後半より): 赤道付近で遠心力の影響を除いても重力が小さいという事実は、赤道方向に質量の過剰、または極方向に質量の不足があることを示唆する。地球の場合、自転による遠心力で赤道半径が極半径より大きくなり赤道バルジを形成するが、それとは別に、マントル対流などによる質量の偏りや、地球よりも顕著な赤道方向へのマントル物質の集中 (より扁平な形状) があるかもしれない。あるいは、マントルが地球よりも流動性が高く、自転の影響をより強く受けて形状が変形しやすいのかもしれない。

地球との比較とまとめ: 惑星Xは地球と同様に層構造を持ち、液体状態の外核を持つ点で似ている。しかし、その液体核は惑星Xのサイズに対して地球よりも相対的に大きく、惑星全体の平均密度が小さいことから、内部を構成する物質の組成が地球と異なる (より軽い元素が多い) 可能性がある。また、赤道付近の重力異常は、マントルのダイナミクスや形状が地球とは異なる特徴を持つことを示唆している。
